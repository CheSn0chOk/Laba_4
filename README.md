# Laba_4
Описание проекта
В ходе выполнения лабораторной работы была разработана полнофункциональная десктопная система управления кассой вокзала на Java Swing. Приложение представляет собой портирование консольной программы в Windows Forms с сохранением всей исходной функциональности и добавлением графического интерфейса.

Основной функционал
Реализовано минимально 4 функциональные вкладки
Реализована выгрузка данных из файла и сохранение в файл
Обеспечено добавление, изменение и удаление данных с визуальным отображением изменений
Реализована сортировка данных по различным критериям
Выполнена проверка полей на правильность ввода и другие логические проверки
Сохранен весь исходный функционал консольного приложения
